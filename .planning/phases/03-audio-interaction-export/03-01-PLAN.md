  <action>
Create playback infrastructure:

**playbackManager.ts** - Singleton Web Audio playback state machine:
1. Class PlaybackManager with private AudioContext, AudioBufferSourceNode | null, AudioBuffer | null
2. Track playback timing: startTime (AudioContext.currentTime), pausedAt (seconds into buffer)
3. setBuffer(buffer: AudioBuffer) - stores buffer, resets pausedAt to 0
4. play() - creates new AudioBufferSourceNode, starts at pausedAt offset, connects to destination
5. pause() - calculates pausedAt from currentTime - startTime, stops source
6. stop() - disconnects and nulls source node
7. seek(time: number) - sets pausedAt, restarts playback if currently playing
8. getCurrentTime() - returns pausedAt if paused, else calculated from currentTime - startTime
9. Export singleton instance (do not create new AudioContext on every import)

**playbackStore.ts** - Zustand store for playback state:
1. State: isPlaying (boolean), playheadTime (number), duration (number)
2. Actions: setPlaying, setPlayheadTime, setDuration
3. Export usePlaybackStore

**PlaybackControls.tsx** - React component:
1. Button toggles play/pause using playbackManager
2. When play clicked: call playbackManager.play(), set isPlaying true, start requestAnimationFrame loop updating playheadTime
3. When pause clicked: call playbackManager.pause(), set isPlaying false, cancel animation frame
4. Display play icon (▶) or pause icon (⏸) based on isPlaying state
5. Disable button when audioStore.outputBuffer is null

Style in app.css:
- Large circular button (64px diameter) with hover state
- Center icon, accessible keyboard focus
  </action>
