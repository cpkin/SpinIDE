  <name>Task 1: Create JSON Schema for metadata v1</name>
  <files>schemas/metadata-v1.schema.json</files>
  <action>
Create `schemas/metadata-v1.schema.json` using JSON Schema 2020-12. Requirements:
- `$schema` set to 2020-12, include `$id`, `title`, and `type: object`.
- Required fields: `effectName`, `io`, `pots`, `memory`, `graph`.
- `version` optional, enum of `v1` and `v2`, default `v1`.
- `io` enum: `mono_mono`, `mono_stereo`, `stereo_stereo`.
- `pots`: array of 3 objects with `id` (pot0/pot1/pot2) and `label` (string, 1-32 chars).
- `memory`: array of objects with `name` and `samples` (integer >0, <=32768).
- `graph`: object with `nodes` (array of ids) and `edges` (from/to ids, allow cycles).
- `additionalProperties: false` at top-level and for sub-objects unless explicitly required.
Include descriptions for each field to support tooling.
  </action>
  <verify>
Schema file exists and is valid JSON; required fields and enums match the action spec.
  </verify>
