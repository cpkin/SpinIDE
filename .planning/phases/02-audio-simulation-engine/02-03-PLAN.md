  <files>src/audio/renderSimulation.ts, src/audio/renderTypes.ts</files>
  <action>
    Create a renderSimulation API that:
    - accepts compiled instruction slots, IO mode, and pot values
    - decodes/resamples input to 32 kHz
    - renders output via OfflineAudioContext so the final AudioBuffer is produced by offline rendering
    - enforces a default 30s render length and hard cap at 120s with warnings
    - emits progress updates once renders exceed 5 seconds
    - supports cancelation via an AbortSignal-style token
    - normalizes output to -1 dB on completion
    Wire the interpreter into this pipeline so the output buffer is ready for playback.
  </action>

  <done>Render pipeline returns output AudioBuffer plus progress and limit warnings.</done>
